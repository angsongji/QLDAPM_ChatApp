# QLDAPM_ChatApp

Clone dự án:

```bash
git clone https://github.com/angsongji/QLDAPM_ChatApp.git
cd QLDAPM_ChatApp
```

# Backend
```bash
cd backend
npm install
```
Sau đó, cấu hình biến môi trường Backend ở (backend/.env)
Rồi chạy backend:
```bash
npm run dev
# server chạy ở http://localhost:5001
```

# Frontend
```bash
cd frontend
npm install
```
Rồi chạy frontend:
```bash
npm run dev
# client chạy ở http://localhost:5173
```
